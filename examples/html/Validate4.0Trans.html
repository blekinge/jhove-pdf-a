This is a test document which conforms to HTML 4.0 Transitional. It has been tested with BBEdit and iCab. It has some features which are intentionally unusual and may cause warning messages, but which are strictly legal. The equivalent file is not valid under HTML 4.0 strict. As far as I can tell, any document which is valid under 4.0 transitional is also valid under 4.01 transitional with an appropriate change of doctype.

It isn't intended to look good.
